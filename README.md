# ⚡ Cyberpunk Linux Desktop Setup

A clean, futuristic cyberpunk-style desktop built on Linux Mint with Conky, Starship, and custom theming.

## 🖼️ Preview

![Preview](preview.png)

---

## ✨ Features

- Minimal cyberpunk Conky panel (CPU, RAM, Disk, Time)
- Neon blue + purple theme
- Starship-powered terminal prompt
- Clean layout (no clutter, focused UI)

---

## 🧩 Components

- **Conky** – system monitor panel
- **Starship** – terminal prompt
- **Zsh** – shell
- **Papirus Dark** – icons
- **JetBrains Mono Nerd Font** – font

---

## 📁 Config Files

- `conky/conky.conf`
- `starship/starship.toml`
